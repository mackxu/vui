# Toast 文字提示
## 概述

一个简短的 toast ，用来反馈用户相关操作的提示。支持自定义位置和持续时间。

## API

### props

| 属性     | 说明                                     | 类型   | 默认值                         | 可选值 |
| :------- | :--------------------------------------- | :----- | :----------------------------- | :----- |
| message  | 显示的文本内容                           | String | 空                             | \      |
| duration | 持续时间（毫秒），若为 -1 则不会自动关闭 | Number | 3000 ms（TV）、3000 ms（手机） | \      |
